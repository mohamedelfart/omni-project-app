# OmniRent Philosophy

- **OmniRent is a real-estate operating system, not a listings app**
- Everything is a request (viewing, maintenance, reservation, services)
- System is stage-based, not page-based
- Command Center is the control surface
- Brain System is the decision layer
- UI is not the system, behavior is the system
- Services appear based on tenant stage
- No direct state mutation outside core
